# Agency Dashboard

Multi-client tracking dashboard for marketing agencies. Centralized master view with individual client isolation.

## Features

- **Master Agency View**: See all clients, metrics, and performance at a glance
- **Client Preview Mode**: Preview what individual clients see (isolated to their data)
- **Dark/Light Mode**: Full theme support with smooth transitions
- **Performance Charts**: Trend lines, donut charts, and key metrics visualization
- **Pagination**: Clean table navigation with consistent height
- **Industry Breakdown**: View clients grouped by industry with aggregated stats
- **Performance Leaderboards**: Rankings by ROAS, CPA, Show Rate, and Revenue

## Metrics Tracked

- Ad Spend
- Appointments Generated
- Cost per Appointment (CPA)
- Return on Ad Spend (ROAS)
- Show Rates (with visual progress bars)
- Revenue per Client
- Qualified Lead Rates

## Tech Stack

- Vanilla HTML/CSS/JS (single-file mockup)
- CSS Variables for theming
- SVG charts (no external dependencies)

## Usage

Open `index.html` in a browser. No build step required.

## Screenshots

Dark mode with client table, summary cards, and pagination. Click the eye icon on any client row to see detailed charts and metrics.
